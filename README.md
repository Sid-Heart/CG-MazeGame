# CG-MazeGame
A Simple Maze Game Using Open GL

# **Controls** 
- W/w - UP 
- S/s - DOWN 
- A/a - LEFT 
- D/d - RIGHT 
- R/r - Reset 
- N/n - NewGame
- U/u - Raise Level 
- L/l - Lower Level
- '+' - Zoom
